Markdown是由约翰·格鲁伯(John Gruber)在2004年创造的一种轻量级文本标记语言。
它允许人们
“使用易读写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。
借助于它，烦琐的格式设置被极大简化，写作者可以更加专注于内容的创作。
格式转换非常方便，根据需要轻松转换为HTML或者PDF文件，
Markdown的语法简便易学，常用的标记符号不超过十个，快速搞定各种排版样式
目前支持Markdown语法的编辑器也非常多，涵盖主流的操作系统，IOS设备也有相当多的APP支持Markdown语法编辑。
许多网站也支持Markdown格式文本直接录入，比如国内的简书，全球最大的轻博客网站Tumblr。
得益于它的简洁好用，Markdown被越来越多的写作爱好者所青睐。
